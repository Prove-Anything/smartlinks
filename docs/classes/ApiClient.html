constructor

  • new ApiClient(baseURL, apiKey?, bearerToken?): ApiClient

  • Creates an instance of ApiClient.

    Parameters

    • baseURL: string

      The base URL of the Smartlinks API (e.g., https://smartlinks.app/api/v1)

    • Optional apiKey: string

      (Optional) API key for X-API-Key header

    • Optional bearerToken: string

      (Optional) Bearer token for AUTHORIZATION header

    Returns ApiClient

    Example

    // With both API key and bearer token
    const client = new ApiClient(
      'https://smartlinks.app/api/v1',
      'your-api-key',
      'your-bearer-token'
    );

    // With only API key
    const client = new ApiClient(
      'https://smartlinks.app/api/v1',
      'your-api-key'
    );

    // With only bearer token
    const client = new ApiClient(
      'https://smartlinks.app/api/v1',
      undefined,
      'your-bearer-token'
    );
